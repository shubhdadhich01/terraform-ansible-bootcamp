
# Enterprise Azure Platform Engineering Bootcamp
## Build Production-Grade Cloud Infrastructure using Terraform & Ansible

> **Duration:** 7 Days  
> **Level:** Intermediate → Advanced → Architect  
> **Format:** Daily Enterprise Use Case + Hands-on Project + Troubleshooting + Interview Preparation

---

# Course Overview

This bootcamp is designed for engineers who want to move beyond basic Azure, Terraform, and Ansible tutorials and learn how enterprise platform teams actually design, automate, secure, govern, and operate production Azure environments.

Instead of isolated labs, participants build one continuous enterprise platform for a fictional organization (**Northwind Retail**). Every day extends the previous day's infrastructure exactly like a real enterprise rollout.
